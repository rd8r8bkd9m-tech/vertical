# Вертикаль — Product Spec

## 1. Обещание продукта

Вертикаль помогает строителю на объекте быстро пройти путь от замера до профессионального предложения клиенту.

Основной сценарий:

```text
Заявка → Замер → Честная цена → Защищённая смета → КП → Документы → Работы → Акт → Оплата
```

## 2. Первый MVP

Первая вертикаль: **механизированная штукатурка**.

MVP обязан закрывать:

- клиент;
- объект;
- замер;
- площадь стен;
- средний слой;
- основание;
- материал;
- расход смеси;
- количество мешков;
- работа;
- подъём;
- логистика;
- антидоп checklist;
- внутренняя экономика;
- PDF КП и смета — следующий этап.

## 3. Уникальные фичи

### Честная цена

Цена делится на:

- работы;
- материалы;
- расходники;
- логистика;
- риск-резерв;
- маржа.

### Антидоп

Система проверяет спорные места:

- слой;
- откосы;
- маяки;
- грунтовка;
- подъём;
- доставка;
- материал чей;
- скрытые дефекты.

### Внутренняя экономика

Клиент видит итоговую цену. Мастер видит маржу и себестоимость.

## 4. Принцип AI

AI не считает деньги. AI разбирает текст, заполняет поля и объясняет результат. Финальный расчёт выполняет C23 core.
