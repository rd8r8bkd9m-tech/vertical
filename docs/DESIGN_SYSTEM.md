# Design System

## Visual direction

Профессиональное мобильное приложение уровня банковского B2B-продукта для строителей.

## Palette

- Primary: `#005BFF`
- Primary dark: `#071A3A`
- Background: `#F5F7FB`
- Surface: `#FFFFFF`
- Success: `#16A34A`
- Warning: `#F59E0B`
- Danger: `#EF4444`
- Text muted: `#64748B`

## Components

- Button
- Input
- Card
- PriceCard
- RiskCard
- ChecklistCard
- StatusBadge
- BottomNav
- ActionSheet
- EstimateItemCard

## UX rule

На объекте: большие кнопки, минимум полей, одна рука, цена сразу.
В офисе: таблицы, версии, документы, аналитика.
