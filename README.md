# Вертикаль

**Вертикаль** — мобильное приложение для строительных мастеров, бригад и компаний: замер, честная цена, защищённая смета, документы и внутренняя экономика.

Главный сценарий:

```text
Заявка → Замер → Быстрая цена → Точная смета → КП → Договор → Работы → Акт → Оплата
```

Первый продуктовый модуль: **механизированная штукатурка**.

## Принцип

- AI помогает разобрать текст и подсказать риски.
- Финальные деньги считает deterministic ядро.
- C23 core является судейским расчётным контуром.
- UI и API написаны на TypeScript.

## Старт разработки

```bash
pnpm install
pnpm dev
```

## Проверка

```bash
pnpm lint
pnpm typecheck
pnpm test
pnpm build
```
